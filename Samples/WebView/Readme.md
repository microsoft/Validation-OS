## Overview
This application demonstrates how to host WebView2 as a simple browser, with the ability to load the WebView2 runtime from a specific directory on the system or the default runtime location.

## How it works?
The application is a basic browser window that uses WebView2 as its web content renderer and processor.
1. Open the solution file in Visual Studio.
2. Select a target architecture and rebuild the solution.
3. Copy the built executable and WebView2 SDK DLL from the bin folder.
4. Double-click the executable to launch the sample application.
