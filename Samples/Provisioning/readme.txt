Name:
    Disk Partition / Provisioning Sample

Description:
    This sample demonstrates how to retrieve the drive letters for a specified disk drive using the Win32_LogicalDiskToPartition WMI Class

Languages:
    cpp

Products:
    windows-api-win32

Build instructions:
    1. Open Provisioning.sln in Visual Studio 2019 or newer with the Windows or Validation OS SDK installed
    2. Build solution

Sample origin:
    This sample is based on a tutorial posted at https://learn.microsoft.com/en-us/windows/win32/cimwin32prov/win32-logicaldisktopartition
