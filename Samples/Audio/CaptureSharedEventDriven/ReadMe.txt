Name:
    WASAPI Capture Shared Event Driven Sample

Description:
    This sample demonstrates capturing audio data using WASAPI.
    The sample captures audio in shared mode using the event driven programming model, saving the captured data to a file.

Languages:
    cpp

Products:
    windows-api-win32

Build instructions:
    To build the sample using the command prompt:
     1. Open the Command Prompt window and navigate to the directory.
     2. Type msbuild WASAPICaptureSharedEventDriven.sln
    
    To build the sample using Visual Studio 2019 or newer (preferred method):
     1. Open Windows Explorer and navigate to the directory.
     2. Double-click the icon for the WASAPICaptureSharedEventDriven.sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory

To run the sample:
    Type WASAPICaptureSharedEventDriven.exe at the command line

Sample origin:
    This sample is based on the samples provided with the Windows 7.1 SDK https://www.microsoft.com/en-us/download/details.aspx?id=8442
