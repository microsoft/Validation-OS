Name:
    Media Engine DComp Win32 Sample

Description:
    This is a sample Win32 desktop application written in C++ which demonstrates media playback using the MediaFoundation IMFMediaEngine API and DirectComposition.

Languages:
    cpp

Products:
    windows-api-win32

Build instructions:
    1. Open src/MediaEngineDCompWin32Sample.sln in Visual Studio 2019 or newer with the Windows or Validation OS SDK installed
    2. Build solution

Sample origin:
    This sample is based on the MediaEngineDCompWin32Sample posted at https://github.com/microsoft/media-foundation/tree/master/samples/MediaEngineDCompWin32Sample
