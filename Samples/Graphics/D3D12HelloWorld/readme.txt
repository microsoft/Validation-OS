Name:
    Direct3D 12 Hello, World! samples

Description:
    This collection of samples act as an introduction to Direct3D 12. Each sample introduces an elementary concept, and lays a foundation for all of the Direct3D 12 samples.

Included samples:
    * Hello, window! sample
      This sample shows you how to create a window, Direct3D device (with debug layers enabled), and present to the window. These are the basic elements that every sample uses.

    * Hello, triangle! sample
      This sample shows you how to draw a static triangle using a vertex buffer.

    * More 'Direct3D 12 Hello, World!' samples are available at https://github.com/microsoft/DirectX-Graphics-Samples/tree/master/Samples/Desktop

Languages:
    cpp

Products:
    windows-api-win32

Build instructions:
    1. Open src/D3D12HelloWorld.sln in Visual Studio 2019 or newer with the Windows or Validation OS SDK installed
    2. Build solution

Sample origin:
    This sample is based on the samples posted at https://github.com/microsoft/DirectX-Graphics-Samples/tree/master/Samples/Desktop
